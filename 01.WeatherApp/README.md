## How to run app locally

1. install the dependency

```bash
#go to the 01.WeatherApp
cd 01.WeatherApp

#install dependency
yarn install

```

2. copy `.env.example` as `.env` and fill up the data

```bash
#openweathermap API key
VITE_API_KEY=
```

3. Now run the app

```bash
yarn run dev
```
